Dominate

让

基于 Tauri v2 + Rust 的 Windows 桌面自动化工具。通过快捷指令 + 可视化积木理念，将时间触发、系统事件与 20+ 种动作类型自由组合。

Features

核心特性

从触发到执行，从可视化编辑到脚本扩展，覆盖完整的自动化链路。

多类型触发器

Cron 定时、系统事件（开机/USB/网络/进程）、课表触发、手动按钮。覆盖教室与日常全场景，让自动化无处不在。

20+ 种动作功能

应用启动、媒体控制、系统电源、Toast 通知、控制流（if/loop/变量）、Lua 脚本，均开箱即用。

可视化积木编辑

基于 React Flow 的三栏编辑器，拖拽积木块组合动作链，支持条件分支、循环、变量传递。

容错与日志

每动作可配容错策略（继续/停止/回滚/通知），执行日志持久化到 SQLite，便于回溯与审计。

Lua 脚本扩展

内置 mlua 引擎（Lua 5.4），严格沙箱隔离，支持脚本市场直连 GitHub 仓库浏览与安装。

便携式部署

数据与程序同目录，NSIS 安装包分发，支持 U 盘导入导出配置，适配学校机器环境。

Security

沙箱即安全

所有 Lua 脚本运行在严格沙箱中，危险 API 自动拦截。下方演示将自动切换三个场景。

backup.lua Sandbox Active 
-- USB 插入自动备份
-- 严格沙箱：禁用危险 API

local backup_dir = "D:\\" .. os.date("%Y%m%d")
os.mkdir(backup_dir)

local files = {
  "C:\\Users\\me\\Documents\\重要文档",
  "C:\\Users\\me\\Desktop\\笔记"
}

local count = 0
for _, src in ipairs(files) do
  local name = string.match(src, "([^\\]+)$")
  local dst = backup_dir .. "\\" .. name
  if os.copy(src, dst) then
    count = count + 1
    log.info("已备份: " .. name)
  end
end

toast:show("备份完成 · " .. count .. " 个文件")
return { success = true, count = count }

FAQ

常见问题

Dominate 是什么？+

Dominate 是一款面向 Windows 的桌面自动化管理工具，定位为"个人自动化助手"。借鉴 iOS 快捷指令设计理念，通过拖拽积木块组合动作链，配合多种触发器实现日常场景的自动化管理。

支持哪些操作系统？+

仅支持 Windows 10/11，x64 架构。需要全局管理员权限运行（UAC 已关闭）。不考虑跨平台，专注于 Windows 生态深度集成。

项目是开源的吗？+

完全开源，基于 MIT 协议发布。源代码托管在 GitHub，欢迎 Star、Issue、PR。

Lua 脚本安全吗？+

默认严格沙箱，禁用 os.execute / io.popen / loadfile 等危险 API。用户可在设置中开启宽松沙箱（自负风险）。所有 Lua 脚本默认 10 秒超时。上方"沙箱即安全"区块有实时演示。

数据存储在哪里？是否便携？+

数据存储在程序安装目录下的 data/dominate.db（SQLite）。便携式部署，支持 U 盘导入导出配置（.dominate 格式，含 JSON 数据 + Lua 脚本）。

当前进度如何？+

Phase 1（核心调度 + 动作执行）已完成，15 个 unit tests 全部通过。Phase 2（UI 骨架 + 首页 Dashboard）开发中。完整计划见 SPEC.md。
